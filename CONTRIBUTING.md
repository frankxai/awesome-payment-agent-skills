# Contributing to Awesome Payment Agent Skills

Thanks for helping build the best list of resources for giving AI agents safe spend authority.

## Guidelines

- **Safety first.** This list is about paying *safely*. Favor entries that authorize, gate, or audit — not just ones that move money faster.
- **One addition per pull request.** Easier to review, easier to discuss.
- **Keep links live.** No dead links, no tracker redirects in the list itself. Link the official spec, repo, or docs — never an invented deep link.
- **Match the format.** Use the existing table/bullet style of the section you're adding to.
- **Say why it belongs.** One short clause on what makes the entry worth a reader's time.
- **Protocols & standards:** include owner + license + the specific role it plays (authorization vs settlement vs network). Note maturity honestly — beta is beta, scaffold is scaffold.
- **Verify claims.** No fabricated stats, no overstated maturity. If you can't source it, don't claim it.

## What doesn't get merged

- Tools that move money without any authorization, gating, or audit story.
- Closed, dead, or vaporware projects (mark existing ones if they lapse).
- "Spend faster" tooling with no safety surface.
- Anything that encourages giving an agent standing, uncapped access to funds.

Open a PR. Be honest. Keep agents from spending what they weren't authorized to. That's the whole bar.
